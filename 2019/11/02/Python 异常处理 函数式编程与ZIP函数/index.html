函数式编程

高阶函数:接受一个或多个函数作为输入,输出一个函数。

lambda 参数：返回值

map(function, iterable, …)

根据提供的函数对指定序列做映射,Python 3.x 返回迭代器。

reduce(function, iterable[, initializer])

函数会对参数序列中元素进行累积。

filter(function, iterable)

用于过滤序列，过滤掉不符合条件的元素。

ZIP函数

作用：将可迭代的对象作为参数，将对象中对应的元素打包成一个个元组，然后返回由这些元组组成的对象。
