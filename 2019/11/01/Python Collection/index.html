前言

collections模块在这些内置数据类型的基础上，提供了几个额外的数据类型：
1、defaultdict: 带有默认值的字典
2、namedtuple(): 生成可以使用名字来访问元素内容的tuple子类
3、deque: 双端队列，可以快速的从另外一侧追加和推出对象
4、Counter: 计数器，主要用来计数
5、OrderedDict: 有序字典

defaultdict

defaultdict类的初始化函数接受一个类型作为参数,当所访问的键不存在的时候，可以实例化一个值作为默认值

命名元组namedtuple

命名元组是一种易于创建、轻量、不可变的对象类型，与普通元组兼容。
特点：
1、能够迭代
2、能够用索引来访问数据
3、能够通过属性名来访问数据

双端队列Deque

deque GIL是线程安全的，List是非线程安全的

Counter

用于追踪值的出现次数

OrderedDict

OrderedDict会根据放入元素的先后顺序进行排序

ChainMap

用来将多个map组合到一起，可以来模拟嵌套的情景而且多用于模板之中，支持普通map的所有操作
